## ğŸ“ Learning Outcomes

This project demonstrates proficiency in the following key areas:

- **Semantic HTML5 Architecture**: Proper use of semantic tags (`<nav>`, `<article>`, `<section>`, `<aside>`) to ensure accessibility and SEO optimization.
- **Advanced CSS Layouts**:
  - **CSS Grid**: Used for the complex masonry blog layout and the 6-step roadmap alignment.

- **CSS Variables & Theming**: Maintained a consistent design system using root variables for colors, fonts, and spacing.
- **Modern UI Effects**: Created glassmorphism overlays, smooth transitions, and hover states without external libraries.
